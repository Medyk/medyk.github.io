Podłączenie dysku CIFS

  • Dodać dysk z menu z wyłączoną opcją enabled.
  • Otworzyć shell serwera PVE
  • Wykonać następujące polecenia
$ pvesm set backupserver -smbversion 2.0
$ pvesm set backupserver -disable 0
